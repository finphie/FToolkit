# FToolkit

各種処理を詰め合わせたライブラリです。

## 説明

FToolkitは、システムやファイル操作などの各種処理を詰め合わせたライブラリです。

> [!Important]
> 作者が必要な機能のみ実装しています。

> [!Warning]
> メジャーバージョンアップ以外でも破壊的変更を行うことがあります。
